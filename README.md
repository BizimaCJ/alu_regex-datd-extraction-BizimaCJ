Regex-based data extraction program

Description
This program uses regular expressions (regex) to find and extract specific types of data from text.

Data Types Extracted
1. Email addresses
2. URLs (web links)
3. Phone numbers (US format)
4. Times (12-hour and 24-hour format)
5. Hashtags

How It Works
1. The program looks through text for patterns
2. Each pattern is validated to make sure it's real
3. Valid items are displayed, invalid ones are rejected
